INFORMATION MANAGEMENT:
Information management is an emerging field that is concerned with:
The infrastructure used to collect, manage, preserve, store and deliver information.
NETWORKING 1
Computer networking refers to interconnected computing devices that can exchange data and share resources with each other.
These networked devices use a system of rules, called communications protocols, to transmit information over physical or wireless technologies.
WEB SYSTEMS AND TECHNOLOGIES
Web information system, or web-based information system, is an information system that uses Internet web technologies to deliver information and services, to users or other information systems applications.
SYSTEMS INTEGRATION AND ARCHITECTURE 1
The Systems Integration Architecture provides a bridge between the heterogeneous operational applications and platforms.
This architecture ties together the mix of platforms, operating systems, transports, and applications.
